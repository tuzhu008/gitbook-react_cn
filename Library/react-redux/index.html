1. React Redux中文文档

React与 Redux. 的官方绑定方案

有效、灵活.

1.1. 安装

React Redux 需要 React 0.14 或者更高版本.

npm install --save react-redux

假设您使用的是 npm 包管理器，其中有一个模块打包器，比如Webpack 或者 Browserify 以使用CommonJS 模块。

如果您还没有使用npm或现代模块打包器，并且更喜欢使用一个单文件UMD构建，使您可以使用一个全局对象，那么您可以从cdnjs中获取一个预构建版本。对于任何严肃的应用程序，我们都不推荐这种方法，因为大多数与Redux互补的库只在npm上可用。

1.2. React Native

从React Native 0.18起，React Redux 5.x可以与React Native一起工作。如果你有任何关于React Redux 5.x在React Native上问题，运行npm ls react并你没有将 React 重复安装到你的node_modules。我们建议您最好使用npm@3.x来避免这些问题。

如果您使用的是较早版本的React Native，由于 这个问题，则需要继续使用React Redux 3.x 版本 和文档
